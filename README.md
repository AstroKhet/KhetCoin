KhetCoin
--------

KhetCoin is a minimalistic clone of early Bitcoin, letting you explore 
the basics of a blockchain without spending any money.

Installation
------------

1. Clone this repository:

       git clone https://github.com/AstroKhet/Khetcoin.git

2. Change into the project directory:

       cd KhetCoin

3. Create a virtual environment in the project folder:

       python -m venv .venv

4. Activate the virtual environment:
   - Windows:

         .venv\\Scripts\\activate
   - Mac/Linux:

         source .venv/bin/activate

5. Install dependencies:

       pip install -r requirements.txt

6. Run the application:

       python main.py
