Little Endian:
- Block headers (version, prev_hash, merkle_root, timestamp, bits, nonce)
- Transaction data (version, lock_time, sequence numbers)
- Script integers (e.g., pushed data in scripts)
- Varints (variable-length integers like tx input/output counts)
- Hashes in local storage (txids, block hashes stored as byte arrays)
- Hashes in network messages (e.g., inv, getdata, headers, block, tx messages)
- Integers in P2P messages (e.g., version, services, nonce)

- IP addresses (IPv4/IPv6 in network messages like addr/version)
- Port numbers (in network messages, e.g., addr/version)

- Network protocol numbers (e.g., port 8333 in code, converted to BE for messages)
- Display/UI formatting of hashes (user-facing hex strings reverse LE storage)
